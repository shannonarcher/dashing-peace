# Dashing Peace

Dashing Peace is a digital version of the board game `Onitama` designed to be played by an AI.

## Status

In Progress

## Installation

Use npm to install and run `dashing-peace`.

```
npm i
```

## Usage

Start the application using 
```
npm start
```

### Move a Pawn
```
mv <grid coord> <card name> <card position [1-n]>
```

### Pass Turn
```
pass <card name>
```
